# Reference: Environment Variables

> Complete reference for all environment variables, from `apps/app/src/config.ts` EnvSchema.

> **TL;DR:** After running `qiforge --init`, you only need to add your `OPEN_ROUTER_API_KEY`. Everything else is pre-filled.

**Source key:**

- **CLI** = auto-generated by `qiforge --init` ‚Äî don't edit unless you know what you're doing
- **Manual** = you must fill this in yourself
- **Optional** = customize if needed, but works fine with defaults

---

## General

| Variable      | Required | Default       | Source       | Description                            |
| ------------- | -------- | ------------- | ------------ | -------------------------------------- |
| `NODE_ENV`    | No       | `development` | **Optional** | `development`, `production`, or `test` |
| `PORT`        | No       | `4000`        | **Optional** | Server port                            |
| `ORACLE_NAME` | Yes      | ‚Äî             | **CLI**      | Oracle display name                    |
| `CORS_ORIGIN` | No       | `*`           | **Optional** | Allowed CORS origins                   |
